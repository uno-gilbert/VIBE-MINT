# VibeMint Frontend — Starter

Sepolia 배포한 `VibeMintNFT` 컨tract 주소만 연결하면 동작하는 최소 민팅 UI입니다.

## 설정

```bash
cp .env.example .env
# VITE_CONTRACT_ADDRESS=0x... 편집
npm install
npm run dev
```

Chrome + MetaMask (Sepolia)에서 http://localhost:5173
