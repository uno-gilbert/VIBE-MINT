# VibeMint Frontend — Solution

워크숍 **완성 예제** DApp. supply 표시, whitelist mint, Sepolia 전환, OpenSea/Etherscan 링크 포함.

## 실행

```bash
cp .env.example .env
# VITE_CONTRACT_ADDRESS=0x... (Sepolia 배포 주소)
npm install
npm run dev
```
